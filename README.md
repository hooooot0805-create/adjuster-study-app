# Personal Study PWA

端末内のJSONファイルを初回に読み込み、ブラウザだけで学習できる個人用PWAです。

## Features

- オフライン対応
- 端末内JSONを初回ファイル選択で読み込み
- 教材データはこのリポジトリに含めない
- 読み込んだ教材データはIndexedDBに保存
- 学習進捗はlocalStorageに保存
- ログインなし
- クラウド同期なし
- 外部APIなし

## How To Use

1. アプリをブラウザで開きます。
2. 初回セットアップ画面で `教材データを読み込む` を押します。
3. 端末内に保存した教材JSONを選択します。
4. 読み込み後、学習モードと分野を選んで学習します。
5. 次回以降は、同じブラウザであれば保存済みの教材データを使って起動します。

## Study Modes

- 穴埋め4択
- 複数穴埋め
- 原文確認
- 生成問題
- 暗記カード
- 苦手復習
- 10問 / 25問 / 50問の簡易テスト

## Data Storage

- 教材データ: IndexedDB
- 学習進捗: localStorage
- アプリ本体: Service Worker cache

ブラウザのサイトデータを削除すると、読み込んだ教材データと学習進捗は消えます。その場合は、教材JSONを再度読み込んでください。

## App Updates

画面下部の管理エリアから以下を実行できます。

- アプリ更新を確認
- アプリを再読み込み
- 教材データ再読み込み
- 教材データ削除
- 進捗リセット

アプリ本体の更新と教材データの再読み込みは別操作です。

## Privacy

教材データはリポジトリに含めません。通常利用時、アプリは教材JSONをネットワークから取得しません。選択した端末内ファイルをブラウザ内に保存して利用します。

## Local Development

```bash
python3 -m http.server -b 127.0.0.1 5173
```

```text
http://127.0.0.1:5173/
```

ローカル開発時だけ、必要に応じて `?dev=1` を付けるとローカル配置した教材JSONの自動読み込みを確認できます。本番公開では教材JSONを配置しないでください。

```text
http://127.0.0.1:5173/?dev=1
```
